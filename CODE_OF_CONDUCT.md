# Code Of Conduct

Be respectful, precise, and constructive.

Investor Harness is built for serious research workflows. Discussions should focus on reproducibility, evidence quality, safety, and maintainability.

Unacceptable behavior includes harassment, abusive language, doxxing, publishing private credentials, or presenting unsupported financial claims as fact.

Project maintainers may remove content or block participation when behavior makes the project unsafe or unproductive.
