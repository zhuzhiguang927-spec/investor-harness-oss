# SM Hourly Watch

这个 skill 解决的是"股票池太大，盘中没法每只都盯"的问题。目标不是做分钟级交易，而是把**真正值得你人工介入的盘中变化**筛出来。

## 强制流程（v0.1 硬约束）

> ⛔ **任何分析输出之前**，必须严格执行 [`../../core/preamble.md`](../../core/preamble.md) 的 6 步开始前流程
>
> ⛔ **任何输出完成之前**，必须严格执行 [`../../core/postamble.md`](../../core/postamble.md) 的 8 步结束后流程
>
> 输出归档按 [`../../core/output-archive.md`](../../core/output-archive.md) 命名规范
> 输出验收按 [`../../core/acceptance.md`](../../core/acceptance.md) 清单逐条自检
>
> **跳过任何一环视为未完成任务。**

Hourly Watch 特别注意：

- **只在连续交易时段判断盘中异动**；集合竞价、午休和收盘后默认不报盘中结论
- **必须比较相对上一轮的变化**，不能只看当前绝对涨跌幅
- **红灯一定要有触发原因**：价格加速、成交放大、公告、新闻、板块共振，至少命中一个
- **背离一定要和同赛道对比**：板块强、个股弱，才叫背离

## 适用场景

- "每小时看一下我这组票有没有异动"
- "盯一下光模块 / PCB / 算力链盘中有没有强弱分化"
- "只告诉我该人工看的几只，不要全量念一遍"
- "盘中有异动时，先告诉我是价格问题、量能问题还是公告问题"

## 不适用场景

- ❌ 1 分钟 / 5 分钟短线交易信号
- ❌ 盘口逐笔 / Level 2 交易型监控
- ❌ 期货、期权、外汇

## 监控框架

### 第一步：定义本轮与上轮

```
[Preflight - Hourly Watch]
本轮时间：{YYYY-MM-DD HH:MM}
比较基准：上一轮 {HH:MM}
股票池：{N} 只
赛道：光模块 / CPO / PCB / 算力链
重点告警：涨停/接近涨停、放量加速、逆势背离、公告催化
```

### 第二步：四层扫描

1. **价格层**
   - 当前涨跌幅
   - 相对上一轮的涨跌幅变化（加速 / 钝化 / 反转）
   - 是否接近涨停 / 跌停

2. **量能层**
   - 当前成交额 / 换手
   - 相对上一轮是否明显放量
   - 量价关系：量价齐升 / 放量滞涨 / 缩量回落

3. **事件层**
   - 当轮是否出现公告、新闻、交易所异动说明
   - 如无新增事件，要明确写"未见新增公开催化"

4. **板块层**
   - 同组核心票平均表现
   - 强势主线是光模块、PCB、交换机还是算力芯片
   - 个股是跟涨、领涨，还是逆势掉队

### 第三步：只分两类

#### 红灯

命中任一条件即可进入红灯：

- 接近涨停 / 涨停
- 相对上一轮 30-60 分钟**明显加速**
- 放量突破关键位
- 叠加公告 / 新闻 / 异动说明
- 板块共振下成为领涨核心

#### 背离

必须同时满足：

1. 同赛道整体偏强或至少中性偏强
2. 该股票逆势下跌、平盘明显弱于同组，或冲高回落显著弱于龙头

## 输出格式

```markdown
# Hourly Watch · {YYYY-MM-DD HH:MM}

**比较区间**：{上轮 HH:MM → 本轮 HH:MM}
**赛道背景**：{一句话，说明本轮谁强谁弱}

## 红灯

- 华工科技（000988.SZ）｜本轮 +3.2pct 至 +9.6%，成交额较上轮放大 45%｜接近涨停 + 光模块板块共振
- 罗博特科（300757.SZ）｜本轮由 +7.1% 加速到 +11.8%｜放量加速，叠加盘中异动事件

## 背离

- 中天科技（600522.SH）｜本轮由 -0.4% 扩大到 -1.9%｜光模块主线偏强但自身继续走弱
- 长飞光纤（601869.SH）｜本轮平盘转 -0.8%，显著跑输同组｜未见新增公开催化，属于弱跟随
```

如果本轮没有命中，直接输出：

```markdown
本轮无红灯/背离
```

## 约束

- ❌ 不要把所有上涨票都报出来
- ❌ 不要只写"因为 AI 板块强"这种空原因
- ❌ 不做交易建议，不给买卖点
- ✅ 只报**需要人工注意**的变化
- ✅ 没有新增公开催化时要明说"数据不足 / 未见新增公开催化"
- ✅ 板块背景必须交代一句
