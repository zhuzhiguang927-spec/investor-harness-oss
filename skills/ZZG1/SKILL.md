# ZZG1 行业研究 · 10 段框架

## 概述

ZZG1 是 ZZG (个股研究) 的赛道版，输出 **10 段行业研究报告** + **7.1 财务对比表**
+ **7.2 市场份额** + **7.3 头部公司核心竞争力**。报告结构与定性风格对齐
IMA 笔记本 `Industry Analysis` 里的 **PCB 行业研究报告** 作为 golden template
(见下方"Golden Template")。

**7.1 财务对比表 = `industry_data.py` + `zzg1_build_table.py` 处理的产物**。
所有 1-14 列数据由 `industry_data.py` 一次性批量拉取，禁止手工估算。
第 15 列 (主营业务/收入占比) 由 Agent 联网整理后通过 `--business` JSON 传入。

## 工作流

```
Step 1  标的池筛选 (Agent 用 cn-web-search / 浏览器搜索 + 业务筛选)
Step 2  python zzg1_make_input.py --industry <行业> --codes <codes> --out zzg1_input.txt
Step 3  python industry_data.py --input zzg1_input.txt --out zzg1_data.json
         (1-14 列数据: 25A 财务 / 行情 / 26-28E 一致预期 / Forward PE / CAGR / PEG / 卖方样本数)
Step 4  Agent 联网整理 7.1 表第 15 列 (主营业务/收入占比, 一句话业务描述) → 存为 business.json
Step 5  python zzg1_build_table.py --input zzg1_data.json --business business.json --out 7_1.md
         (输出 15 列 7.1 表, 按 PEG 升序排)
Step 6  写 10 段报告 (其中 7.1 = 7_1.md 表格嵌入, 7.2/7.3 联网整理)
Step 7  同步归档到 outputs/ + themes/<行业>/ + Desktop\AI Analysis\Industry Analysis\
Step 8  上传 IMA Industry Analysis 笔记本 (folder_id: {IMA_INDUSTRY_ANALYSIS_FOLDER_ID})
```

## 标的池筛选规则 (Step 1 必做)

### A 股行业本体筛选
- 只含 A 股上市 (沪深主板/创业板/科创板/北交所)，**剔除港股/美股/未上市**
- 只含 **行业本体公司**，**剔除** 上下游/材料/设备/概念股/代工配套
- 按总市值降序；有多少拿多少，**上限 10 家** (不要为凑数拉边缘公司)
- 个别公司数据找不到的，写"找不到"或"不可算"，**不强行估算**

### 头部公司选择标准 (按 PCB 报告口径)
按 "**收入规模 / 行业链敞口 / 利润弹性 / 制造壁垒 / 上市公司可比性**" 5 维选 5-10 家。
例: PCB 行业头部 10 家 = 东山精密 / 鹏鼎控股 / 深南电路 / 胜宏科技 / 沪电股份 /
景旺电子 / 生益电子 / 兴森科技 / 崇达技术 / 广合科技。

### 上下游剥离规则 (按 PCB 报告口径)
**不纳入 7.1 头部表**的公司:
- 上游材料: CCL、铜箔、树脂、玻纤布、电子化学品
- 下游应用: 终端品牌、组装厂
- 设备厂商: PCB 设备、CCL 设备
- 概念股 / 题材股
- 7.1 头部公司剔除的上下游公司，**只在第 3 章产业链 + 第 6 章价格/利润传导 讨论**，
  不进入第 7 章公司分析
- 例: 生益科技、南亚新材、金安国纪、华正新材、方邦股份、中英科技、逸豪新材 (CCL/材料)
  在 PCB 报告里被剔除头部表，只在产业链和价格传导章节讨论

## Forward PE 与 PEG 口径 (硬约束)

### Forward PE
- **口径**: 2026年6月5日总市值 / 2026E盈利预测
- 取不到可靠 2026E 预测时，**不用历史市盈率口径替代**
- 写"不可算"，不强行估算

### 26E PEG (传统 PEG 公式, 按用户原话)
- **公式**: **26E PEG = 26E Forward PE / 净利润 CAGR** (传统 PEG = PE / G)
- **CAGR 数据源**: 优先 `data_agent.fetch_market_akshare` 一致预期净利均值；
  一致预期缺失时取近 2 个月卖方研报盈利预测均值

#### CAGR 按拉到几年预测分 4 种情况 (用户原话: "有几年算几年, 算几何平均")

26E 几乎肯定能拿到 (一致预期必有); 25A 一般都有; 27E/28E 经常缺失。

| 拉到几年 | CAGR 公式 | 表格标注 | cagr_years |
|---|---|---|---|
| 25A + 26E + 27E + 28E 全有 | `(NP_28E / NP_25A) ^ (1/3) - 1` | `47.30%` (无标注) | 3 |
| 25A + 26E + 27E 有, 28E 缺失 | `(NP_27E / NP_25A) ^ (1/2) - 1` | `35.20% (2Y)` | 2 |
| 只有 25A + 26E, 27E 和 28E 都缺失 | `NP_26E / NP_25A - 1` (单年同比) | `55.50% (单年同比)` | 1 |
| 25A 或 26E 缺失 (一致预期和研报都没有) | 不可算 | `不可算` | 0 |

- **禁止**: 把 25A 实际增速混入 CAGR 预测
- **本表 PEG 数值是第 10 章 10.0-10.5 象限判断的唯一输入**
- **⚠️ 口径差异**: PCB 报告原文里写的 PEG 公式是 `CAGR / Forward PE` (反 PEG),
  ZZG1 统一用 `Forward PE / CAGR` (传统 PEG), 两者数值相反。
  例: 鹏鼎 26E PE 45.4x, CAGR 39.7% -> ZZG1 PEG = 1.14, PCB 报告 PEG = 0.87 (反)。

## 第 7.2 章 市场份额与竞争地位

**份额估算口径** (按 PCB 报告):
- 全球 PCB 产值分母: 2025 年约 858 亿美元，折合人民币约 6178 亿元
- 中国大陆产值分母: 占全球 60%，约 3707 亿元
- 某公司全球份额 = 公司 PCB 相关收入 / 6178 亿
- 某公司国内份额 = 公司 PCB 相关收入 / 3707 亿

**份额趋势分 4 档**:
- 强升: AI 服务器/封装基板/高阶 HDI 弹性最大
- 平台偏稳: FPC/消费电子 PCB 龙头
- 平台缓升: 多场景 PCB 供应商
- 修复 / 转型验证: 客户结构变化中

**不参与份额排名的特殊情况**: PCB 精确收入未披露的公司 (如东山精密)
只列业务进展，不参与份额排名。

## 第 7.3 章 头部公司核心竞争力分析

每家头部公司 **1-2 段**，按四要素:
1. **竞争力强/中强/中等** (定性判断)
2. **核心优势** (客户结构 / 工艺壁垒 / 资本开支 / 上下游协同)
3. **核心短板** (毛利率压力 / 客户集中度 / 产能爬坡风险)
4. **变化趋势** (在变强 / 正在变强 / 修复 / 转型验证)

末段给小公司群分析 (按利润率 / 客户类型 / 高端料号占比分组)。

## 第 10 章 PEG × 护城河 二维矩阵

按 PEG 分组 + 护城河强弱 (高/中/低) 划分 4 象限:

| 象限 | PEG 范围 | 护城河 | 结论 |
|---|---|---|---|
| 1. PEG 合理 + 高护城河 | 0.9-1.4 | 高 | **最强 alpha** (胜宏/沪电/鹏鼎/深南) |
| 2. PEG 偏低 + 高护城河 | <0.9 | 高 | 价值投资候选 (鹏鼎 0.87x) |
| 3. PEG 偏高 + 高护城河 | 1.4-2.0 | 高 | 估值偏贵，需护城河支撑 (生益电子 1.33x) |
| 4. PEG 失真 (PE 极高) | 任意 | 中/低 | 价值陷阱候选 (兴森 PE 151.6x, 25A 基数低) |
| 5. 持续亏损 PCB 小厂 | 不可算 | 任意 | 7.1 之外，参考价值有限 |

每家公司必须给:
- PEG 数值
- 护城河强弱判断 (高/中/低)
- 26E PE 绝对水平
- 在矩阵中的象限
- 核心担忧 + 关键反转观察点

## 10 段报告结构 (对齐 golden template)

1. **行业概览** —— 行业是什么 + 核心功能 + 下游 + 近年关键变化 (例: 2023-2025 行业核心变化)
2. **行业增长分析** —— 总市场规模 + 增速 + 周期 vs 成长 + 未来 5 年增长驱动
3. **产业链结构层级** —— 上游材料 / 中游基材 / 中游制造 / 下游应用 + 各环节议价能力 (用表格列: 主要环节/代表公司/价值量与利润分布/议价能力)
4. **需求侧驱动** —— 哪些下游/技术/政策在驱动 + 各驱动量化贡献
5. **供给侧变化** —— 产能扩张/收缩 + 高低端分化 + 关键供给瓶颈
6. **价格/利润传导** —— 上游成本 → 中游报价 → 下游接受度 + 传导周期
7. **重点公司和竞争力分析** —— 7.1 财务表 (15 列) + 7.1 后小公司补充表 (5 列) + 7.2 市场份额 + 7.3 头部公司核心竞争力
8. **当前市场争议点** —— 多空双方分歧 + 关键变量 + 验证点
9. **最值得跟踪的行业指标** —— 5-10 个量化指标 + 跟踪频率 + 数据源
10. **投资建议 (行业级)** —— PEG × 护城河矩阵 + 看多/看空组合 + 风险提示

## 数据来源（分两层：定性来源 + 定量脚本）

### A. 定性数据来源（继承 Investor Harness 来源链）

**强制要求**：
- **不看摘要原则**：禁止只读搜索 snippet、评级、目标价、PE 表或盈利预测摘要；必须读正文 / PDF。
- **港股 / 美股行业**用 iFind MCP `global_stock` 工具拉数据。

- **事实锚 vs 市场观点**：
  - 事实锚：公司公告、年报、财报全文、iFind MCP 拉取的可追溯披露材料
  - 市场观点：卖方报告中的评级、目标价、PE、盈利预测 —— 只作市场观点，**不得当作事实**
- **缺口披露**：如果近 5 年行业研报不足、无法打开、只有摘要，最终报告必须明确列入"仍需补的资料 / 资料缺口"。
