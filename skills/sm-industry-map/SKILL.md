## 输出重点 — 8 段固定骨架（v0.4 强化）

> ⛔ **行业研究最终交付物必须严格按以下 8 段展开，缺一段、调换顺序、合并小节、改写标题都视为未完成。**
>
> 这 8 段是**不可压缩的固定骨架**——不是建议结构。
> 行业研究的好坏差异，**主要在第 6 段「重点公司与分工」的展开深度**。

### 1. 行业一句话判断

用一句话给出对行业方向、节奏、当前位置的核心判断。允许追加 1 句"目前处在 X 阶段、Y 拐点、Z 验证窗口"。

### 2. 产业链结构

**必须显式拆上、中、下游**，每层至少给出：

- 该层的**主要环节**（生产什么 / 提供什么服务）
- **代表公司**（2-3 家，含中外）
- **价值量分布**（哪一层吃掉了产业链利润的大头）
- **议价能力强弱**（面对上游/下游时是 price maker 还是 price taker）

### 3. 需求侧驱动

拆**主要下游应用**（如 AI 服务器 / 800G 交换机 / 新能源车 / 消费电子等），每个下游回答：

- **驱动力来源**：产品代际、技术升级、政策、库存周期、宏观周期 等
- **量化量级**：下游增速、ASP、单机价值量、渗透率

### 4. 供给侧变化

回答**供给端正在发生什么**，包括：

- 玩家进出（新进入者、退出者、并购重组）
- 产能投放节奏（在建 / 投产 / 爬坡 / 满产）
- 海外布局（东南亚 / 墨西哥 / 欧洲）
- 价格战 / 紧缺切换
- **份额转移方向**（从谁流向谁）

用**具体数字**说话，不写"供给紧张"这种空话。

### 5. 价格 / 利润传导

画出**传导路径**：

> 上游成本（原材料 / 设备 / 能源）→ 加工费 / 报价 → 公司毛利率

并明确：

- 哪些环节**议价权强**（能转嫁）
- 哪些环节**被压缩**（吃不到利润）
- 关键变量（如铜价、油价、汇率、某型号紧缺度）对哪一段影响最大

### 6. 重点公司与分工 ⭐（本 skill 最重要的输出）

**至少覆盖 5 家关键公司**，每家都按以下 **3 个子段**展开。**这是本 skill 区分"行业研究"和"公司研究"的核心，必须深挖。**

#### 6.1 财务与经营对比

**横向对比**所有覆盖公司，给出可比表 + 关键发现。

**必含字段**：

- 收入（YoY）
- 归母净利润（YoY）
- 毛利率
- 净利率
- 研发费用率
- 管理费用率
- 销售费用率
- 总市值（最新交易日）
- **PE：当前 TTM PE**（用最新年报/季报净利口径，不引用卖方一致预期，不预测）

**多元化业务公司必须显式拆分行业收入占比**：

- 例：东山精密虽属 PCB 行业，但 PCB 收入只是其总收入的一部分，必须**显式算出"PCB 业务收入 / 公司总收入"**才能正确归入 PCB 行业格局。
- 例：胜宏科技若同时做 PCB 和 IC 载板，必须分别列示两块收入。
- 拆分依据：年报分部数据、卖方研报分部拆分、公司投资者交流纪要；找不到时标注"未披露"，不编造。

**关键发现段**：

- 谁规模最大、谁最赚钱、谁增长最快
- 费用率结构的差异（如某家研发费用率显著高于同行——是技术驱动还是费用失控？）
- PE 估值差异的可解释因素（增速差 / 利润率差 / 业务结构差）

#### 6.2 市场份额与竞争地位

**必须分两个口径**：

1. **全球份额**（全球行业产值 / 出货量为分母）
2. **国内份额**（国内行业产值 / 出货量为分母）

**必须量化到 1% 精度**（如 7% / 2% / 35%，不接受"份额领先"这种词）。

**必须判断趋势**：

- 最近 3-5 年份额是**升 / 平 / 降**？
- 趋势变化的方向（强升 / 缓升 / 平台 / 缓降 / 强降）
- 变化背后的可观测驱动（产能投放 / 客户突破 / 收购 / 出货量数据）

**估算口径必须显式标注"估算"二字**——不需要给公式或假设，但要让读者知道这个数字是算出来的而不是查到的。

**还应附**：公司公开披露 / 卖方研报中的全球排名（如 Prismark / N.T.Information / CPCA 排名），区分"公司自报"vs"第三方机构口径"。

#### 6.3 核心竞争力分析 ⭐⭐（**本 skill 最重要的子段**）

**逐家公司回答"这家公司在这个行业里凭什么赢"**——这是用户最看重的部分。

**核心要求**：明确说清楚每家公司的竞争优势**到底在哪**。要展开分析，不能泛泛而谈"技术领先"、"客户优质"这种空话——必须把优势的**具体表现**、**强度**、**可持续性**讲明白。

**展开方式**（按行业实际情况挑维度，不强求覆盖全部）：

- **技术**：技术领先多少代 / 多少年、壁垒多高、是否可被绕过、研发投入是否可持续、关键专利布局
- **客户**：核心客户名单与粘性、认证周期、客户切换成本、是否进入海外 Tier 1 客户体系
- **品牌 / 渠道**：在终端品牌中的认知度、议价权、渠道布局深度
- **产能**：产能规模 / 利用率 / 爬坡进度 / 海外占比 / 单位投资强度
- **规模效应 / 资本开支**：单位成本曲线位置、capex 强度、回收周期
- **认证壁垒 / 资质**：行业认证（汽车 IATF 16949 / 医疗 / 航天 / 通信 NEBS 等）覆盖度
- **海外交付 / 本地化能力**：海外工厂、关税规避、本地团队

> **写多深、写哪些维度，行业不同答案不同**——PCB 行业看产能 / 客户 / 高层板工艺；半导体设备看技术代际 / 客户验证；消费品看品牌 / 渠道 / 供应链管理。**以"能不能解释清楚竞争优势"为准，不要为了凑维度而写空话。**

**最后给两个判断**（每家公司都需明确）：

- **竞争力强不强**（强 / 中 / 弱）
- **竞争力有没有变化**（变强 / 持平 / 变弱），并给出驱动因素

### 7. 当前市场争议点

列出 **2-3 个真正的市场争议**——**不是"前景广阔"这种共识**，而是**卖方之间、买方之间、产业内不同玩家之间有真实分歧**的点。

每个争议必须给出：

- **多方观点**（A 派 vs B 派 vs C 派，引用具体机构 / 卖方 / 产业人士）
- **可观测验证指标 / 时点**（什么数据 / 事件出现意味着哪一派赢）

### 8. 最值得跟踪的行业指标

至少 **3 个**可量化、可定期更新的指标。每个指标写明：

- 指标名称 + 量化定义
- **更新频率**（日 / 周 / 月 / 季）
- **数据来源**（哪个机构 / 数据库 / 行业协会）
- **为什么重要**（与产业链哪一段、哪条投资逻辑挂钩）

## 约束

- **8 段骨架不可压缩、不可调换、不可改写标题**。少一段、不按顺序、合并小节都视为未完成。
- 不要只列公司名单；第 6 段每家公司都要有"为什么"和"凭什么"。
- 不要把宏观背景写得过长（≤ 200 字以内）。
- 最终要落到"**谁受益、谁受损、何时验证**"——给具体时间锚点（季度 / 月度 / 特定事件）。
- **禁止把标签化来源分级写进正文**（见 `core/adapters.md` 总原则 #3 + `core/acceptance.md` B5）；只在后台做可靠性判断。
- **禁止单独的"资料缺口"、"来源分级"、"研究边界"、"参考来源"独立章节**（见 `core/acceptance.md` C2）；缺口在正文相邻位置短注，弱来源短标。
- 末尾**不堆后台信息**（输出时间、Task ID、工具流程等），只放归档提示和合规一句话。
- 最终报告归档后必须执行 IMA 上传步骤，并在最终回复确认已上传到 IMA `Industry Analysis`；不要输出 `note_id`，除非用户明确要求。
