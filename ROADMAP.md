# Roadmap

## Near Term

- Generalize local configuration for all upload and archive paths.
- Add more public example reports with synthetic or fully sourced data.
- Add stronger CI checks for route-table consistency.
- Add Markdown link checking.
- Add a small command-line smoke test for onboarding and skill discovery.

## Medium Term

- Provide reusable templates for additional public data-source adapters.
- Add adapters for multiple data-source stacks.
- Build a public benchmark suite for hallucination-resistance and evidence labeling.
- Improve English documentation for international users.

## Long Term

- Make Investor Harness a general research workflow standard for agentic due diligence, not only secondary-market research.
- Support pluggable archival backends.
- Provide maintained examples for company research, industry research, comparison research, and event-driven opportunity analysis.
